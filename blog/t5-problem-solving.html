Dear 'non-tech friend', here are some definitions for you...

  • .map() is a way of performing an operation of some kind (mathematical, formatting etc) on an array (a type of data structure that you could think of as a chain of boxes each containing some data or groups of data) and storing the answers/outputs in a new array.
  • .filter() is - rather as the name suggests - a way to filter an array for those boxes whose data matches the chosen filtering. The contents of the matching boxes are stored in a new array (chain of boxes).
  • .reduce() provides a way to process the contents of the boxes in relationship to each other. This is a bit different to most methods that work on arrays: normally each box gets processed in turn but independently. It can deliver a single value, for example if each element in the boxes are added together or another array.
